📐 2次方程式

基礎から応用まで分かりやすく学ぼう

📚2次方程式とは

2次方程式とは、未知数 x の2次式を0とおいた方程式のことです。一般的な形は次のように表されます：

ax² + bx + c = 0
（a ≠ 0, a, b, c は定数）

この式において：

  • a：2次の係数（0でない）
  • b：1次の係数
  • c：定数項

🔍解の公式

2次方程式 ax² + bx + c = 0 の解は、以下の公式で求めることができます：

x = (-b ± √(b² - 4ac)) / 2a

この公式の中の b² - 4ac を判別式（discriminant）と呼び、解の性質を決定します：

  • b² - 4ac > 0 ：異なる2つの実数解
  • b² - 4ac = 0 ：重解（1つの実数解）
  • b² - 4ac < 0 ：2つの複素数解

💡解法の例

例題：x² - 5x + 6 = 0 を解いてみましょう

Step 1: 係数を確認
a = 1, b = -5, c = 6
Step 2: 判別式を計算
b² - 4ac = (-5)² - 4(1)(6) = 25 - 24 = 1 > 0
→ 異なる2つの実数解があります
Step 3: 解の公式を適用
x = (5 ± √1) / 2 = (5 ± 1) / 2
x = 3 または x = 2

🧮2次方程式計算機

ax² + bx + c = 0 の形で係数を入力してください

📊2次関数のグラフ

2次方程式 ax² + bx + c = 0 の解は、2次関数 y = ax² + bx + c のグラフがx軸と交わる点のx座標になります。

グラフは上の計算機の係数に基づいて自動更新されます

🎯因数分解による解法

2次方程式は因数分解でも解くことができます。特に係数が整数の場合に有効です。

例：x² - 5x + 6 = 0

x² - 5x + 6 = (x - 2)(x - 3) = 0
したがって、x - 2 = 0 または x - 3 = 0
よって x = 2 または x = 3
